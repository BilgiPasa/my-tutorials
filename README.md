# my-tutorials

İnternette bulduğum bilgilerle yaptığım rehberlerim

Lütfen emin olmadan, (ben dahil) kimsenin yazmış olduğu komutları/yönergeleri direk kullanmayınız. Lütfen kullanmak istediğiniz komutların/yönergelerin ne işe yaradıklarını araştırınız ve emin olmadığınız komutları direk kullanmayınız.

Bu rehberleri kendim için yaptım. Bu rehberlerdeki komutların/yönergelerin sizde problem çıkarıp çıkarmayacağı konusunda garanti veremem. Ayrıca, her rehberin çalıştığının garantisini de veremem.
