# my-tutorials

İnternette bulduğum bilgilerle yaptığım tutorial'larım

Lütfen emin olmadan, (ben dahil) kimsenin yazmış olduğu komutları/yönergeleri direk kullanmayınız. Lütfen kullanmak istediğiniz komutların/yönergelerin ne işe yaradıklarını araştırınız ve emin olmadığınız komutları direk kullanmayınız.

Bu tutorial'ları kendim için yaptım. Bu tutorial'lardaki komutların/yönergelerin sizde problem çıkarıp çıkarmayacağı konusunda garanti veremem. Ayrıca, her tutorial'ın çalıştığının garantisini de veremem.
