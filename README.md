# my-tutorials

İnternetten bulduğum bilgilerle yaptığım tutorial'larım

Lütfen emin olmadan, (ben dahil) kimsenin yazmış olduğu komutları/yönergeleri direk kullanmayın. Kullanmak istediğiniz komutların/yönergelerin ne işe yaradıklarını araştırın ve emin olmadığınız komutları direk kullanmayın.

Bu tutorial'ları ben kendim için yaptım. Bu tutorial'lardaki komutların/yönergelerin sizde problem çıkarıp çıkarmayacağı konusunda garanti veremem. Ayrıca her tutorial'ın çalıştığının garantisini de veremem. Haberiniz olsun.
