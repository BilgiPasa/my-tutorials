Fedora kuracaksan Fedora Workstation ISO'su veya Fedora KDE ISO'su ile kurulum yapmanı öneririm. Çünkü eğer
ki o ikisi hariç başka bir ISO ile Fedora'yı kurarsan başlangıç penceresi çıkmayabilir ya da çıkarsa
"Enable third party repositories" seçeneği çıkmayabilir. Yani kurulum sonrası 3. parti repoları tek bi tuşla
açmak yerine kendin açman gerekir.

Fedora Workstation kuruyorsan ve kurulumda konum seçmeye geldiysen, oradaki arama kısmında konumunu yazma.
Oradaki haritadan seç ve devam et (harita üstündeki simgenin gitmesi için haritada simgesiz olan kısma tıkla ve
sonra kendi konumuna tıkla ve "Next" de). Çünkü arama kısmına yazıp devam etmeye çalışırsan Fedora donabilir.

Fedora Workstation veya Fedora KDE kuruyorsan "Enable third party repositories" butonuna tıkla ve öyle devam
et (3. parti depoları açman için). Eğer ki bunu açmayı unutup kurulum yaparsan uygulama mağazasında software
repositories kısmından aç.

Ardından; sistem güç modunu "Performance" yap, "Dim Screen" ayarını kapat, ekran kilitleme zamanını ayarla ve
kullanmıyorsan bluetooth'u kapat.

Ardından, sistem güncellemelerini uygulama mağazasından yap ve uygulamada çıkan "Restart" tuşu ile bilgisayarını
yeniden başlat.

Fedora'da dnf'i daha hızlı yapmak için şunu yap
{
    Terminale şu komutu girerek nano'dan bir dosyayı aç: sudo nano /etc/dnf/dnf.conf

    ardından o dosyanın içinde, "[main]" KISMININ ALTINA şu 2 şeyi yaz
    {
        max_parallel_downloads=10
        fastestmirror=True
    }

    ardından kaydet ve kapat.
}

Fedora masaüstü ortamın ne olursa olsun terminale şu komutları gir
{
    sudo dnf update -y
    sudo dnf install fastfetch mesa-libGLU -y
    sudo dnf remove libreoffice* -y
}

GNOME kullanıyorsan terminale şu komutları gir
{
    touch ~/Templates/'Empty text file'.txt
    sudo dnf install gnome-themes-extra -y
}

GNOME kullanıyorsan uygulama mağazasında sağ üstteki 3 çizgili menüden "Software Repositories"i aç. Orada eğer
ki açık olan "Fedora Flatpaks" reposu varsa kapat. (Merak etme. Bu, Flathub reposunu etkilemiyor.)

KDE kullanıyorsan "Discover"ı aç. Settings'e gel. Orada eğer ki açık olan "Fedora Flatpaks" reposu varsa kapat.
(Merak etme. Bu, Flathub reposunu etkilemiyor.)

NVIDIA sürücülerini kuracaksan terminale şu komutları gir (Kaynak : https://rpmfusion.org/Howto/NVIDIA)
(CUDA sürücülerinin adında xorg ve x11 geçiyor ama Wayland'de de çalışıyor yani haberin olsun.)
{
    sudo dnf update -y
    sudo dnf install akmod-nvidia -y
    sudo dnf install xorg-x11-drv-nvidia-cuda -y
}

Sürücüleri kurunca 5 dakika zamanlayıcı kur ve bekle. Ardından bilgisayarını yeniden başlat.

Settings'te tap to click'i aç, mouse acceleration'ı kapat ve mikrofon input sesini kısmayı unutma.

GNOME kullanıyorsan uygulama mağazasından "Extension Manager"ı flatpak olarak kur. Ardından, Extension
Manager'dan bu eklentileri sistemine kur:
    Alphabetic App Grid (stuarthayhurst'ın)
    Appindicator and KStatusNotifierItem Support (3v1n0'nun)
    Blur my Shell (aunetx'in)
    Clipboard Indicator (Tudmotu'nun)
    Extension List (grroot'un)
    Removable Drive Menu (fmuellner'in)
    System Monitor (fmuellner'in)
    Touchpad On Off (r_mano'nun)

Ardından, Extension Manager'da "System Extensions" kısmındaki "Background Logo" eklentisini KAPAT! Yoksa 2 kere
Super tuşuna basıp uygulamaların olduğu yere gelirken animasyon takılabiliyor veya sağ alttaki logo
takılabiliyor.

GNOME'da Blur my Shell eklentisinin ayarlarına gir, altta "Panel" kısmına tıkla. Sonra "Override background"
kısmında daha fazla seçenek çıkar ve "Disable when a window is near" seçeneğini işaretle.

GNOME'da Alphabetic App Grid eklentisinin ayarlarına gir ve "Position of ordered folders" ayarını "End" olarak
ayarla.

GNOME'da, ayarlar uygulamasının "Multitasking" kısmında "Hot Corner" ayarı açıksa kapat.

GNOME kullanıyorsan "GNOME Tweaks"i sistem paketi olarak sistemine kur. Ardından GNOME Tweaks'te
"Middle Click Paste" ayarını kapat ve "Appearance" kısmındaki "Styles" kısmından "Legacy Applications"ı
Adwaita-dark olarak ayarla.

GNOME'da Nautilus'ta (dosya yöneticisi) Desktop klasörünü ve kendi istediğin klasörleri sol menüde kısayol
olarak ayarla. Nautilus'ta Ctrl + , (kontrol + virgül) kısayoluyla "Preferences" menüsünü aç ve orada
"Sort Folders Before Files" ayarını (ilk ayar olması lazım) aç.

GNOME'da uygulamalara büyütme ve küçültme butonları eklemek istiyorsan (ki gerek yok ama istiyorsan
ekleyebilirsin) "GNOME Tweaks"i aç, "Windows" kısmına gel ve "Titlebar Buttons" kısmından "Maximize" ile
"Minimize" ayarlarını aç.

KDE'de Dolphin (dosya yöneticisi) açıldığında home folder'ını açmasının ayarını yap. Dolphin'de arkaplan'a çift
tıkladığında bütün her şeyi seçme ayarını da kapat.

GNOME kullanıyorsan (nokyan'ın) "Resources" uygulamasını flatpak olarak uygulama mağazasından indir.

GNOME kullanıyorsan ayarlarda klavye kısayolları kısmında:
    "Switch windows"u "Alt + Tab" yap.

    "Hide all normal windows"u "Super + D" yap.

    "Resources" adında bir kısayol yap. Kısayolu "Ctrl + Shift + Escape" olsun. Komutu ise
    "flatpak run net.nokyan.Resources" olsun.

    "Nautilus" adında bir kısayol yap. Kısayolu "Super + E" olsun. Komutu ise "nautilus" olsun.

    Eğer ki "Ctrl + Alt + T" komutu terminal açmazsa "Terminal" ismine bir kısayol yap. Kısayolu
    "Ctrl + Alt + T" olsun. Komutu ise o terminalin komutu ne ise o olsun. (Komut; "ptyxis", "gnome-terminal",
    "kgx" veya başka bişey olabilir.)

    İstersen "Switch applications"ı "Super + Tab" yap.
    (İstersen bu keybind'ların hepsini kullanabilirsin ve aşağıda "farklı virtual desktop'lardaki açık
    penceler arasında Alt + Tab yaparak dolaşma"yı açmayabilirsin. Ya da en son keybind'ı girmeyip aşağıda
    "farklı virtual desktop'lardaki açık penceler arasında Alt + Tab yaparak dolaşma"yı açabilirsin.)

GNOME'da eğer ki farklı virtual desktop'lardaki açık penceler arasında Alt + Tab yaparak dolaşmak istiyorsan
(Eğer ki Alt + Tab'ı "Switch windows"a ayarladıysan bunun etkisini görebilirsin.)
{
    gsettings set org.gnome.shell.window-switcher current-workspace-only false
}
Bunu kapatmak istiyorsan
{
    gsettings set org.gnome.shell.window-switcher current-workspace-only true
}

Eğer ki GNOME'da default terminalini değiştirmek istemiyorsan
{
    Terminal uygulamasını aç. Sağ üstteki 3 çizgi menüden "Preferences"ı aç. "Profiles" kısmına gel ve profil
    adını "Main Profile" yap. "Appearance" kısmına gel ve "Palette" kısmında "Show All Palettes"e tıkla. Oradan
    "Argonaut"u seç. "Show Fewer Palettes"e tıkla ve aşağıya in. "Use System Font"u kapat. Custom Font olarak
    "Roboto Mono Regular"ı seç ve font boyutunu 16 yap (ve eğer ki Roboto Mono çıkmıyor ise Roboto Mono fontunu
    sistemine yükle ve ardından bunu yap). Cursor'dan "Cursor Shape"i "Underline" yap. "Behaviour" kısmına gel
    ve "Interface" kısmında "Restore Session"ı ve "Restore Window Size"ı kapat. "Restore Window Size" kısmından
    çıkan yerdeki seçenekleri fastfetch'e göre ayarla. ("Coloumns"u 97, "Rows"u 29 yap. Olmazsa değişirirsin.)
    Sonra "Behaviour" menüsünde en aşağıya in ve "Set as Default Terminal"e tıkla.
}
Eğer ki GNOME'da default terminalini değiştirmek istiyorsan
{
    Eğer ki "Desktop Icons NG" isimli GNOME extension eklentisine uygun bir terminal uygulaması istiyorsan,
    "gnome-terminal" olan terminali sistemine kurabilirsin.

    Sonrasında şu siteden https://github.com/Stunkymonkey/nautilus-open-any-terminal sitedeki uygulamayı kur
    (ve haberin olsun komutların başında sudo yazmıyorsa komutların başına sudo yazman lazım). Ardından
    yönergelere uyarak default terminali yapmak istediğin terminal yap (Dconf Editor ile yaptım ben. Dconf
    Editor uygulamasını flatpak olarak sistemine kurabilirsin). Terminal uygulamasını klayve kısayoluyla
    açmak için ayarlardan yapmış olduğun klavye kısayolu ayarını değiştirmeyi unutma. Terminalin adı "Terminal"
    veya "Console" ise komutları "gnome-terminal", "kgx" veya "ptyxis" olabilir. Ve eğer ki Nautilus'ta
    "Open Terminal Here" diye bir şey yazmıyorsa bilgisayarını yeniden başlat.
}

Eğer ki saatini ve formatlarını Türkiye'ye göre uygun ama İngilizce yapmak istiyorsan
{
    GNOME'da, ayarlarda "System" kısmında gir ve orada "Region and Language" kısmına gir. "Language"ı
    "English (United States)" seç, "Formats"ı "United Kingdom (English)" seç. Sisteminden çıkıp geri giriş
    yapınca etki gösterir.

    KDE'de ayarlarda Region and Language ayarı olması lazım. Oraya gir, dil olarak Amerikan İngilizcesini seç,
    formatlar olarak ise "C" seç. Sisteminden çıkıp geri giriş yapınca etki gösterir.
}

Eğer ki NVIDIA ekran kartı kullanıyorsan ve Wayland'de problemler yaşıyorsan X11'e geçebilirsin ama ilk önce
Wayland'e şans ver. Bi bak. Olmazsa X11'e geçmek için şunları yap
{
    KDE'de
    {
        İlk önce şu komutu gir: sudo nano /etc/sddm.conf

        Ve dosya içinde "[General]" kısmında "DisplayServer" ile ilgili ayarı "DisplayServer=x11" yap.

        Ardından şu komutu gir: sudo dnf install plasma-workspace-x11 -y

        Ve bilgisayarını yeniden başlat.

        Ardından sistemine giriş yapmak için şifreni girmeden öncesinde sol üstte (orda yoksa başka bir yerde)
        masaüstü ortamı seçeneği olarak Plazma (veya Plazma (Wayland)) değil, Plazma (X11)'i seç.

        Ardından şifreni girip sistemine giriş yap.

        Sonrasında şu komutu gir: sudo dnf install vdpauinfo libva-vdpau-driver libva-utils -y

        Eğer ki komut çalışmaz ve "libva-vdpau-driver" paketini bulamama ile ilgili mesaj yazarsa bu komutu
        gir: sudo dnf install vdpauinfo libva-utils -y

        Ardından şu "https://github.com/JoseExposito/touchegg" siteden Fedora'ya göre kurma yönergelerine
        uyarak touchegg'i sistemine kur.

        Bilgisayarı yeniden başlattıktan sonra Fedora, mouse acceleration açık gibi çalışıyor olabilir. Eğer
        ki mouse acceleration açık gibi hissediyorsan mouse acceleration'ı aç kapa yap.

        Taskbar'a (alttaki uzun çubuk) sağ tık yap ve "Show panel configuration"a tıkla. NVIDIA GPU ile X11
        kullanmak, şu şu şeylerde sıkıntı çıkarabilir gibi bi yazı olması lazım. Onun yanındaki "Fix it"
        butonuna tıkla ve fix edince "Exit edit mode" butonuna tıkla.
    }
    GNOME'da (GNOME 49'dan itibaren artık bunu yapamayabilirsin.)
    {
        İlk önce şu komutu gir: sudo dnf install gnome-session-xsession -y

        Ardından bilgisayarını yeniden başlat.

        Sistemine giriş yapmak için kullanıcını seç ve şifreni girmeden öncesinde sağ alt kısımdaki (orada
        yoksa başka bir yerdeki) dişli çark görünümlü butona tıkla ve seçenek olarak "GNOME" yerine
        "GNOME on Xorg" seçeneğini seç ve şifreni girerek sistemine giriş yap.

        Ardından şu komutu gir: sudo dnf install vdpauinfo libva-vdpau-driver libva-utils -y

        Ardından şu "https://github.com/JoseExposito/touchegg" siteden Fedora'ya göre kurma yönergelerine
        uyarak touchegg'i sistemine kur.

        Sonra Fedora'nın uygulama mağazasından "Extension Manager"ı flathub şekliyle yükle ve Extension
        Manager'dan "X11 Gestures" eklentisini yükle.
    }

    Kaynak : https://www.if-not-true-then-false.com/2015/fedora-nvidia-guide/7/
}
