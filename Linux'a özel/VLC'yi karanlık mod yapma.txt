Eğer ki bilgisayar teman karanlıksa ama VLC'yi açtığında VLC karanlık tema değilse ilk önce VLC'yi kapat.

Sonrasında
{
    Debian kullanıyorsan terminale şu komutu gir
    {
        sudo apt install adwaita-qt
    }

    Fedora kullanıyorsan terminale şu komutu gir
    {
        sudo dnf install gnome-themes-extra
    }
}

Sonrasında "Tools" kısmından, "Preferences"ı aç ve "Force Window Style" kısmından karaklık tema seç.
