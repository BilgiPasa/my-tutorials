Sistemin Debian tabanlı ise terminale şu komutu gir:
    sudo apt install vim

Sistemin Fedora tabanlı ise terminale şu komutu gir:
    sudo dnf install vim

Sistemin Arch Linux tabanlı ise terminale şu komutu gir:
    sudo pacman -S vim

Debian'ın vim ile alakalı sitesi: https://wiki.debian.org/vim
Fedora'nın vim ile alakalı sitesi: https://fedoraproject.org/wiki/Vim
Arch Linux'un vim ile alakalı sitesi: https://wiki.archlinux.org/title/Vim
