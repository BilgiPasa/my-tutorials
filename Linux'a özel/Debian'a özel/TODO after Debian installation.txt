To install NVIDIA drivers
{
    (if you have a problem in the solution that I am going to give, go to the
    https://wiki.debian.org/NvidiaGraphicsDrivers website and find your solution.)

    Enter this command to the terminal: sudo apt install linux-headers-amd64

    Then enter this command to the terminal: sudo nano /etc/apt/sources.list

    and change this line
    {
        deb http://deb.debian.org/debian/ trixie main non-free-firmware
    }

    to this line
    {
        deb http://deb.debian.org/debian/ trixie main contrib non-free non-free-firmware
    }

    and save.

    Then enter this command to the terminal: sudo apt update && sudo apt upgrade

    Then enter this command to the terminal: sudo apt install nvidia-kernel-dkms nvidia-driver firmware-misc-nonfree

    If there is no problem, reboot the system and select X11 session when you are logging in.
}

When installing, there can be a pop up. Accept the pop up. Then reboot your pc.

If you use GNOME; in keyboard shortcuts:
    Make "Switch windows" as "Alt + Tab".
    Make "Hide all normal windows" as "Super + D".
    Make a "System Monitor" shortcut as "Ctrl + Shift + Escape". The command is "gnome-system-monitor".
    Make a "Nautilus" shortcut as "Super + E". The command is "nautilus".
    If "Ctrl + Alt + T" does not open a terminal, make a custom shortcut named "Terminal" and make its command
    as what your terminal command is. It is either "ptyxis", "gnome-terminal", "kgx" or something else.
    If you want, you can make "Switch applications" as "Super + Tab".
    (İstersen bu keybind'ların hepsini kullanabilirsin ve aşağıda "farklı virtual desktop'lardaki açık
    penceler arasında Alt + Tab yaparak dolaşma"yı açmayabilirsin. Ya da en son keybind'ı girmeyip aşağıda
    "farklı virtual desktop'lardaki açık penceler arasında Alt + Tab yaparak dolaşma"yı açabilirsin.)

GNOME'da eğer ki farklı virtual desktop'lardaki açık penceler arasında Alt + Tab yaparak dolaşmak istiyorsan
(Eğer ki Alt + Tab'ı "Switch windows"a ayarladıysan bunun etkisini görebilirsin.)
{
    gsettings set org.gnome.shell.window-switcher current-workspace-only false
}
Bunu kapatmak istiyorsan
{
    gsettings set org.gnome.shell.window-switcher current-workspace-only true
}

Eğer ki saatini ve formatlarını Türkiye'ye göre uygun ama İngilizce yapmak istiyorsan
{
    GNOME'da, ayarlarda "System" kısmında gir ve orada "Region and Language" kısmına gir. "Language"ı
    "English (United States)" seç, "Formats"ı "United Kingdom (English)" seç. Bilgisayardan çıkış giriş
    yapınca etki gösterir.

    KDE'de ayarlarda Region and Language ayarı olması lazım. Oraya gir, dil olarak Amerikan İngilizcesini seç,
    formatlar olarak ise "C" seç. Bilgisayardan çıkış giriş yapınca etki gösterir.
}

How to make Firefox scrolling smooth:
    At the search bar type "about:config", accept the risks.
    Write "general.smoothScroll.msdPhysics.enabled" in the search preference name menu.
    Set it to "true".
