Bu yapılacaklar listesini, Ubuntu'nun GNOME versiyonu için yaptım.

İlk önce, software updater uygulamasıyla sistemini güncelleştir. Güncelleme bitince gerekiyorsa bilgisayarını
yeniden başlat.

Ardından, terminale şu komutları gir
{
    touch ~/Templates/'Empty Document'.txt
    sudo apt install adwaita-qt curl
    curl -SsL https://playit-cloud.github.io/ppa/key.gpg | gpg --dearmor | sudo tee /etc/apt/trusted.gpg.d/playit.gpg >/dev/null
    echo "deb [signed-by=/etc/apt/trusted.gpg.d/playit.gpg] https://playit-cloud.github.io/ppa/data ./" | sudo tee /etc/apt/sources.list.d/playit-cloud.list
    sudo apt update
    sudo apt install playit
}

Ardından; sistem güç modunu "Performance" yap, "Dim Screen" ayarını kapat, ekran kilitleme zamanını ayarla ve
kullanmıyorsan bluetooth'u kapat.

Settings'te tap to click'i aç, mouse acceleration'ı kapat ve mikrofon input sesini kısmayı unutma.

Flatpak'i şu sitedeki yönergelere uyarak kur: https://flatpak.org/setup/Ubuntu

VSCode'u şu sitedeki yönergelere uyarak kur: https://code.visualstudio.com/docs/setup/linux#_debian-and-ubuntu-based-distributions

Şu siteden Steam'in .deb dosyasını indir ve kur: https://store.steampowered.com/about/

X11 kullanıyorsan "touchegg"i şu sitedeki yönergelere uyarak kur: https://github.com/JoseExposito/touchegg

Eğer ki snap'leri sisteminden kaldırmak istiyorsan ilk önce Firefox'u veya başka ne tarayıcı kullanmak istiyorsan
onu snap olmayacak şekilde sistemine kur. Ardından şu sitedeki yönergeleri takip et:
    https://www.debugpoint.com/remove-snap-ubuntu/

Eğer ki snap'leri kaldırdıysan home directory'sindeki "snap" klasörünü sil.

Uygulama mağazasından "Extension Manager"ı flatpak olarak kur. Ardından, Extension Manager'dan bu eklentileri
sistemine kur:
    Alphabetic App Grid
    Blur my Shell
    Extension List (grroot'un)
    Removable Drive Menu (fmuellner'in)
    Clipboard Indicator (Tudmotu'nun)
    System Monitor (fmuellner'in)
    Touchpad On Off
    Appindicator and KStatusNotifierItem Support

Blur my shell indirirsen o eklentinin ayarlarına gir, altta "Panel" kısmına tıkla. Sonra "Override background"
kısmında daha fazla seçenek çıkar ve "Disable when a window is near" seçeneğini işaretle.

Eğer ki system monitor extension'ı çalışmıyorsa şu komutu terminale gir ve bilgisayarını yeniden başlat:
    sudo apt install gir1.2-gtop-2.0 libgtop2-dev

Settings'te nolur nolmaz "default apps" kısmına bak.

Settings'te "Multitasking" kısmında "Hot Corner" ayarı açıksa kapat.

"GNOME Tweaks"te (indirmediysen sistem paketi olacak şekilde indir) "Middle Click Paste"i kapat ve
"Appearance" kısmındaki "Styles" kısmından "Legacy Applications"ı Adwaita-dark yap.

Nautilus'ta (dosya yöneticisi) Desktop klasörünü ve kendi istediğin klasörleri sol menüde kısayol olarak
ayarla. Nautilus'ta Ctrl + , (kontrol + virgül) kısayoluyla "Preferences" menüsünü aç ve orada
"Sort Folders Before Files" ayarını (ilk ayar olması lazım) aç.

Uygulamalara büyütme ve küçültme butonları eklemek istiyorsan (ki gerek yok ama istiyorsan ekleyebilirsin)
"GNOME Tweaks"i aç, "Windows" kısmına gel ve "Titlebar Buttons" kısmından "Maximize" ile "Minimize" ayarlarını
aç.

Settings'te klavye kısayolları kısmında:
    "Switch windows"u "Alt + Tab" yap.

    "Hide all normal windows"u "Super + D" yap.

    "System Monitor" adında bir kısayol yap. Kısayolu "Ctrl + Shift + Escape" olsun. Komutu ise
    "gnome-system-monitor" olsun.

    "Nautilus" adında bir kısayol yap. Kısayolu "Super + E" olsun. Komutu ise "nautilus" olsun.

    Eğer ki "Ctrl + Alt + T" komutu terminal açmazsa "Terminal" ismine bir kısayol yap. Kısayolu
    "Ctrl + Alt + T" olsun. Komutu ise o terminalin komutu ne ise o olsun. (Komut; "ptyxis", "gnome-terminal",
    "kgx" veya başka bişey olabilir.)

    İstersen "Switch applications"ı "Super + Tab" yap.
    (İstersen bu keybind'ların hepsini kullanabilirsin ve aşağıda "farklı virtual desktop'lardaki açık
    penceler arasında Alt + Tab yaparak dolaşma"yı açmayabilirsin. Ya da en son keybind'ı girmeyip aşağıda
    "farklı virtual desktop'lardaki açık penceler arasında Alt + Tab yaparak dolaşma"yı açabilirsin.)

Eğer ki farklı virtual desktop'lardaki açık penceler arasında Alt + Tab yaparak dolaşmak istiyorsan
(Eğer ki Alt + Tab'ı "Switch windows"a ayarladıysan bunun etkisini görebilirsin.)
{
    gsettings set org.gnome.shell.window-switcher current-workspace-only false
}
Bunu kapatmak istiyorsan
{
    gsettings set org.gnome.shell.window-switcher current-workspace-only true
}

Eğer ki saatini ve formatlarını Türkiye'ye göre uygun ama İngilizce yapmak istiyorsan
{
    Settings'te "System" kısmında gir ve orada "Region and Language" kısmına gir. "Language"ı
    "English (United States)" seç, "Formats"ı "United Kingdom (English)" seç. Sisteminden çıkıp geri giriş
    yapınca etki gösterir.
}

Brave kullanacaksan kurmanı önerdiğim eklentiler:
    Dark Reader, Youtube-shorts block, Return Youtube Dislike

Firefox kullanacaksan kurmanı önerdiğim eklentiler:
    Ublock Origin, Dark Reader, Youtube-shorts block, Return Youtube Dislike, "Dark" isminde bir tema

Firefox'ta akıcı bir şekilde scroll yapabilmek için:
    Firefox'un arama kısmına şunu yaz: about:config
    Riskleri kabul et.
    Oradaki ayar arama kısmına (tırmaksız bir şekilde) "general.smoothScroll.msdPhysics.enabled" yaz.
    O ayarı "true" yap.
