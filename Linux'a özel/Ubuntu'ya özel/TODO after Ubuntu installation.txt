Firstly, update and upgrade your system using the software updater app. If needed, restart your PC.

Then, enter these commands to the terminal
{
    touch ~/Templates/'Empty Document'.txt
    sudo apt install adwaita-qt curl
    curl -SsL https://playit-cloud.github.io/ppa/key.gpg | gpg --dearmor | sudo tee /etc/apt/trusted.gpg.d/playit.gpg >/dev/null
    echo "deb [signed-by=/etc/apt/trusted.gpg.d/playit.gpg] https://playit-cloud.github.io/ppa/data ./" | sudo tee /etc/apt/sources.list.d/playit-cloud.list
    sudo apt update
    sudo apt install playit
    sudo curl -fsSLo /usr/share/keyrings/brave-browser-archive-keyring.gpg https://brave-browser-apt-release.s3.brave.com/brave-browser-archive-keyring.gpg
    sudo curl -fsSLo /etc/apt/sources.list.d/brave-browser-release.sources https://brave-browser-apt-release.s3.brave.com/brave-browser.sources
    sudo apt update
    sudo apt install brave-browser
    sudo snap remove --purge firefox
}

Install Steam from its site (using the .deb package).

Install VS Code by following the instructions at this site: https://code.visualstudio.com/docs/setup/linux#_debian-and-ubuntu-based-distributions

If you use GNOME; in keyboard shortcuts:
    Make "Switch windows" as "Alt + Tab".
    Make "Hide all normal windows" as "Super + D".
    Make a "System Monitor" shortcut as "Ctrl + Shift + Escape". The command is "gnome-system-monitor".
    Make a "Nautilus" shortcut as "Super + E". The command is "nautilus".
    If "Ctrl + Alt + T" does not open a terminal, make a custom shortcut named "Terminal" and make its command
    as what your terminal command is. It is either "ptyxis", "gnome-terminal", "kgx" or something else.
    If you want, you can make "Switch applications" as "Super + Tab".
    (İstersen bu keybind'ların hepsini kullanabilirsin ve aşağıda "farklı virtual desktop'lardaki açık
    penceler arasında Alt + Tab yaparak dolaşma"yı açmayabilirsin. Ya da en son keybind'ı girmeyip aşağıda
    "farklı virtual desktop'lardaki açık penceler arasında Alt + Tab yaparak dolaşma"yı açabilirsin.)

GNOME'da eğer ki farklı virtual desktop'lardaki açık penceler arasında Alt + Tab yaparak dolaşmak istiyorsan
(Eğer ki Alt + Tab'ı "Switch windows"a ayarladıysan bunun etkisini görebilirsin.)
{
    gsettings set org.gnome.shell.window-switcher current-workspace-only false
}
Bunu kapatmak istiyorsan
{
    gsettings set org.gnome.shell.window-switcher current-workspace-only true
}

Check the default apps in the settings just in case.

Extension manager'ı flatpak olarak indir. Sonra Extension manager'ı aç ve oradan activate gnome,
alphabetic app grid, blur my shell, dash to panel, (grroot'un) extension list, removable drive menu,
(Tudmotu'nun) clipboard indicator, (fmuellner'in) system monitor, touchpad on off ve
(X11 kullanıyorsan) X11 Gestures'u indir. Hangilerini kullanmak istiyorsan onları aç ve diğerlerini kapat.

Blur my shell indirirsen o eklentinin ayarlarına gir, altta "Panel" kısmına tıkla. Sonra "Override background"
kısmında daha fazla seçenek çıkar ve "Disable when a window is near" seçeneğini işaretle.

Eğer ki system monitor extension'ı çalışmıyorsa şu komutu terminale gir ve bilgisayarını yeniden başlat:
    sudo apt install gir1.2-gtop-2.0 libgtop2-dev

GNOME'da, ayarlar uygulamasının "Multitasking" kısmında "Hot Corner" ayarı açıksa kapat.

"GNOME Tweaks"te (indirmediysen sistem paketi olacak şekilde indir) "Middle Click Paste"i kapat ve
"Appearance" kısmındaki "Styles" kısmından "Legacy Applications"ı Adwaita-dark yap.

Nautilus'ta (dosya yöneticisi) Desktop klasörünü ve kendi istediğin klasörleri sol menüde kısayol olarak
ayarla. Nautilus'ta Ctrl + , (kontrol + virgül) kısayoluyla "Preferences" menüsünü aç ve orada
"Sort Folders Before Files" ayarını (ilk ayar olması lazım) aç.

Uygulamalara büyütme ve küçültme butonları eklemek istiyorsan (ki gerek yok ama istiyorsan ekleyebilirsin),
"GNOME Tweaks"i aç, "Windows" kısmına gel ve "Titlebar Buttons" kısmından "Maximize" ile "Minimize" ayarlarını
aç.

How to make Firefox scrolling smooth:
    At the search bar type "about:config", accept the risks.
    Write "general.smoothScroll.msdPhysics.enabled" in the search preference name menu.
    Set it to "true".
