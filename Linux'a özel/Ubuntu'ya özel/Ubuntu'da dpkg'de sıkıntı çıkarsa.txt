Bir paketi kurarken sorun çıkarırsa ve "sudo apt upgrade" yapamıyorsan çözümü şu
{
    Terminale şu komutları gir:
        sudo apt update
        sudo apt upgrade

    Şu pakette sıkıntı var tarzı bir yazı çıkabilir, o sıkıntı olan paketin adını unutma.

    Ardından, terminale şu komutu gir:
        sudo dpkg --remove --force-all (sıkıntılı paketin adı)

    Ardından, terminale tekrar şu komutları gir:
        sudo apt update
        sudo apt upgrade
}

Daha fazla bilgi için: https://askubuntu.com/questions/88371/apt-synaptic-needs-to-reinstall-package-but-cant-find-the-archive-for-it
