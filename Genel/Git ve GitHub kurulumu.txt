GitHub hesabın yoksa bir GitHub hesabı oluştur. Ardından, istersen bir GitHub deposu (repository) oluştur.

Eğer ki Windows kullanıyorsan sonraki adımları yapmak yerine GitHub Desktop uygulamasını kullanabilirsin.
(Ama ben kullanmıyorum. GitHub'ı terminalde kullanmayı tercih ediyorum.)

Eğer ki Windows kullanıyorsan Git'i, terminale şu komutu girerek kur: winget install -e --id Git.Git

Eğer ki Linux kullanıyorsan Git, sisteminde zaten yüklü olabilir veya GitHub CLI kurduğunda otomatik
kurulabilir.

Masaütünde (veya başka istediğin bir yerde) "GitHub" isimli bir klasör oluştur. Kullanacağım GitHub depolarını
bu klasöre klonla.

Git'i terminalde kullanmak ve GitHub ile bağlamak için GitHub CLI'ı kur.

Sistemin Debian tabanlı ise şu sitedeki komutları kullanarak GitHub CLI'ı sistemine kur:
    https://github.com/cli/cli/blob/trunk/docs/install_linux.md#debian

Sistemin Fedora tabanlı ise şu sitedeki komutları kullanarak GitHub CLI'ı sistemine kur:
    https://github.com/cli/cli/blob/trunk/docs/install_linux.md#rpm

Windows kullanıyorsan şu sitedeki komutu kullanarak GitHub CLI'ı sistemine kur:
    https://github.com/cli/cli/blob/trunk/docs/install_windows.md#winget

Windows kullanıyorsan GitHub CLI'ı kurduktan sonra terminali kapat ve tekrardan aç.

Ardından, GitHub hesabınla giriş yapmak için terminale şu komutu gir:
    gh auth login

Linux kullanıyorsan giriş yaptığına emin olmak için şu komutu gir:
    git config -l | grep credential

Eğer ki hiçbir yazı çıkmazsa "gh auth login" komutu ile düzgün bir şekilde giriş yapmaya çalış.

Son olarak, terminale bu komutları gir
(<GitHub mailini buraya yaz> kısmına kendi GitHub mailini yaz.)
(<GitHub kullanıcı adını buraya yaz> kısmına kendi GitHub kullanıcı adını yaz.)
{
    git config --global user.email "<GitHub mailini buraya yaz>"
    git config --global user.name "<GitHub kullanıcı adını buraya yaz>"
}

Git config'ine girmiş olduğun maili ve ismi görmek için şu komutu kullanabilirsin:
    git config --list
