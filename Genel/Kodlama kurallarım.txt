Yazdığın kod anlaşılır olsun.

Kod yazacaksan o kodu kendin yaz. (Yani sen kod yazacaksan o kodu makine değil, sen yaz.)

Sevdiğin programları kullan.

Sevdiğin bir font'u kullan. (Ben şahsen Roboto Mono'yu kullanıyorum.)

Gereksiz yere yorum satırı yazma.

Şartlı ifadeler, döngüler ve fonksiyonların öncesine ve sonrasına birer satır boşluk bırak.

Değişken adları, fonksiyon adları, class adları, obje adları, dosya adları ve klasör adları açıklayıcı olsun
ama (alt tireler, kısa çizgiler ve boşluklar dahil) 30 karakteri geçmesin.

Gerektiğinde bir adı kısaltabilirsin.

Ad kısaltmak için bu kısaltmaları kullanabilirsin:
    array -> arr
    camera -> cam
    collider -> coll
    collision polygon -> coll polygon
    collision shape -> coll shape
    degree -> deg
    field of view -> FOV
    frames per second -> fps
    image -> img
    information -> info
    object -> obj
    magnitude -> mag
    manager -> mng
    maximum -> max
    minimum -> min
    multiplier -> mult
    normalized -> normd
    number -> num
    pointer -> ptr
    position -> pos
    radian -> rad
    rigid body -> rb
    rotation -> rot
    static body -> sb
    summation -> sum
    temporary -> temp
    velocity -> vel

    Genel olarak kullanabileceğin başka kısaltmalar olursa buraya yaz. (Sıra alfabetik olarak kalsın.)

Değişkenleri oluştururken tiplerine göre sıralı bir şekilde yaz (C# dilinden örnek):
    byte verySmallNumber;
    short smallNumber
    int number;
    long bigNumber;
    float decimalNumber;
    double bigDecimalNumber;
    bool trueOrFalse;
    char someCharacter;
    string someText;
