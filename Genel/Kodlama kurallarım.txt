Yazdığın kod anlaşılır olsun.

Generitive AI prompt'u (yani ChatGPT, Gemini, Claude vs.) kullanma.

Autocomplete kullanmak istiyorsan kullan.

Sevdiğin bir text editörünü veya IDE'yi kullan. (Ben şahsen VS Code'u kullanıyorum.)

Sevdiğin bir font'u kullan. (Ben şahsen Roboto Mono'yu kullanıyorum.)

Gereksiz yere yorum satırı yazma.

Değişken adları, fonksiyon adları, class adları, obje adları, dosya adları ve klasör adları açıklayıcı olsun
ama 30 karakteri geçmesin (boşluklar, alt tireler ve kısa çizgiler dahil).

Gerektiğinde bir adı kısaltabilirsin.

Ad kısaltmak için bu kısaltmaları kullanabilirsin:
    camera => cam
    collider => coll
    collision polygon => coll polygon
    collision shape => coll shape
    field of view => FOV
    frames per second => fps
    image => img
    information => info
    object => obj
    magnitude => mag
    manager => mng
    maximum => max
    minimum => min
    multiplier => mult
    normalized => normd
    position => pos
    rigid body => rb
    rotation => rot
    static body => sb
    summation => sum
    temporary => temp
    velocity => vel
    Başka kullandığın kısaltmalar olursa buraya yaz. (Sıra alfabetik olarak kalsın.)

Değişkenleri oluştururken tiplerine göre sıralı bir şekilde yaz (C# dilinden örnek):
    byte verySmallNumber;
    short smallNumber
    int number;
    long bigNumber;
    float decimalNumber;
    double bigDecimalNumber;
    bool trueOrFalse;
    char someCharacter;
    string someText;

Enum, array vb. tiplerin değerlerini birden çok sıra halinde yazacaksan son elemanın yanına da virgül koy.
C# dilinden örnek:
    enum Seasons
    {
        Spring,
        Summer,
        Autumn,
        Winter,
    }
(Evet, son elemana virgül koyman gerekmiyor. Ama en son elemana virgül koymak; sonrasında eleman ekleme,
çıkarma ve yer değiştirmeyi daha da kolaylaştırabilir. Ayrıca, Godot da böyle kullanılmasını öneriyor.
https://docs.godotengine.org/en/stable/tutorials/scripting/gdscript/gdscript_styleguide.html#trailing-comma)
