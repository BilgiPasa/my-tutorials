Godot'un .NET halini indirdiğine emin ol. Öyle indirmediysen Godot'u sil ve .NET halini sitesinden indir.

Godot'u ilk açtığında; Godot, İngilizce değilse sağ üstteki ayarlar kısmından dili İngilizce yap.

Eğer ki Godot'un görünümünü beğenmediysen o ayarlar menüsünden "Style"ı "Classic" ve "Color Preset"i
"Godot 3" yapabilirsin. Normal haline geri döndürmek için "Style"ı "Modern" ve "Color Preset"i "Default"
yapabilirsin.

Godot'ta yeni bir proje oluştur veya var olan bir projeye gir.

"Editor Settings"e gir ve "Advanced Settings" ayarını aç.

General kısmında:
    Interface > Editor kısmında:
        Code Font Size -> 20
        "Code Font" yerinde, Godot'un kod editöründe kullanmak istediğin fontun dosya konumunu belirt.

    Text Editor > Completion kısmında:
        Idle Parse Delay -> 0.5
        Idle Parse Delay With Errors Found -> 0.25
        Code Complete Delay -> 0.25

Shortcuts kısmında:
    Arama kısmına "Add Child Node" yaz. Oradaki "Add Child Node" kısayolunun sağındaki artı butonuna tıkla ve
    Shift + A event'ini, "Physical Keycode" olarak DEĞİL, "Keycode" olarak ekle.
    (Keycode olarak eklemeni istememin sebebi, Ctrl + A kısayolu Keycode olarak bulunduğundan dolayı.)
    Bunu yapmamın sebebi, Blender'daki Shift A kısayolu işlevini Godot'ta sağlamak için.

Godot'u VS Code veya VSCodium'da GDScript ile kullanmak istersen:
    VS Code veya VSCodium'u aç. Açtığın programda "godot-tools" eklentisini kur.

    O kurduğun eklentinin ayarlarındaki "Server Port" ayarına gel.

    Godot'ta, Editor Settings > General > Network > Language Server kısmında bulunan "Remote Port" ayarına
    gel ve "Remote Port" ayarındaki yazan sayıyı "Server Port" ayarına yaz.
    (Eğer ki Godot'ta "Language Server" ayarını göremiyorsan "Advanced Settings"i aç.)

    Godot'ta Editor Settings > General > Text Editor > External kısmına gel, "Use External Editor"ı aç.

    VS Code ile kullanmak için "Exec Path"e (tırnaksız bir şekilde) "code" yaz.

    VSCodium ile kullanmak için "Exec Path"e (tırnaksız bir şekilde) "codium" yaz.

    Oraya code veya codium yazıp enter tuşuna bastıktan sonra "Exec Flags"te otomatik olarak
    (tırnaksız bir şekilde) "{project} --goto {file}:{line}:{col}" diye doldurulması lazım. Eğer ki otomatik
    olarak doldurulmadısya o yazıyı oraya yaz.

    Artık Godot'ta script açmaya çalıştığında VS Code veya VSCodium'dan açılması lazım. İlk kez açıldığında,
    Godot'un çalıştırma dosyasının nerede olduğunu belirtmeni isteyebilir. Onu belirttikten sonra, programı
    yeniden başlatma butonu çıkması lazım. Ona tıkla.

Godot'u VS Code veya VSCodium'da C# ile kullanmak istersen:
    VS Code ile kullanmak için Microsoft'un "C#" eklentisi ile "C# Tools for Godot" eklentisini kur.

    VSCodium ile kullanmak için muhammad-sammy'nin "C#" eklentisi ile "C# Tools for Godot" eklentisini kur.

    Godot'ta Editor Settings > General > Dotnet > Editor kısmında "Visual Studio Code" bulunan seçeneği seç
    ve "Custom Exec Path" yerine (tırnaksız bir şekilde) "code" yaz.

Eğer ki Godot'u PascalCase ile kullanmak istersen:
    Project Settings'e gir.

    "Default Signal Callback Name"i (tırnaksız şekilde) "On{NodeName}{SignalName}" yap.

    "Default Signal Callback to Self Name"i (tırnaksız şekilde) "On{SignalName}" yap.

    "Scene Name Casing" ve "Script Name Casing"te "PascalCase" seçeneğini seç.
