VSCodium'u kurduktan sonra VSCodium'u aç ve geri kapat.

Ardından, VSCodium ayar dosyanı olması gereken yere koy.

Ardından, masaüstünden bir text dosyasını VSCodium'da aç. O dosyayı "Restricted Mode"da açtır.

Ardından, alt çubuğun solundaki "Restricted Mode" butonuna tıkla. Çıkan menüde aşağı scroll yap, sol alttaki
"Add Folder" butonunu kullanarak masaüstü klasörün ile D klasörünü oraya güvenilen dosyalar olarak ekle.
Ekledikten sonra, o menüdeki "Trust" butonuna bas ve o menüyü kapat.

Ardından, sol çubuktaki "Extensions" menüsünden şu eklentileri kur:
    BasedPyright (detachhead'in)
    Better Comments (aaron-bond'un)
    Bracket Select (chunsen)
    C# (muhammad-sammy'nin) (Arama kısmına CSharp diye yazarsan çıkabilir.)
    C# Tools for Godot (neikeq'in)
    clangd (llvm-vs-code-extensions'ın)
    Extension Pack for Java (vscjava'nın)
    godot-tools (geequlim'in)
    Material Icon Theme (PKief'in) (Bu eklentiyi kurduktan sonra ikon teması olarak bunu seç.)
    Python (ms-python'un)
    Trailing Spaces (shardulm94'ün)
    Turkish Language Pack for Visual Studio Code (MS-CEINTL'in)

Bu eklentileri kurdukran sonra sağ altta İngilizce olarak "Dili değiştirmek ister misiniz?" tarzında bir
bildirim gelirse "Change Language and Restart" butonuna tıkla ve VSCodium'u yeniden başlat.

Eğer ki "clangd" eklentisini kurduysan, sistemine clangd paketini kur.

Debian tabanlı sistemler için clangd kurulum komutu:
    sudo apt install clangd

Fedora tabanlı sistemler için clangd kurulum komutu:
    sudo dnf install clangd

Eğer ki "Python" eklentisini kurduğunda, yanında "Python Environments" eklentisi de kurulduysa
"Python Environments" eklentisini kaldır.

Ardından, VSCodium'un sol çubuğundaki "Kaynak Denetimi", "Çalıştırma ve Hata Ayıklama", "Godot Tools" ve
"Hesaplar" menülerini gizle.

Ardından, sol çubuktaki "Gezgin" menüsünü aç ve o menünün aşağısındaki "Anahat" ve "Zaman Çizelgesi"
bölümlerini gizle.

Ardından, alt çubuğun solunda "Uzak Ana Bilgisayar" bölümü gözüküyorsa o bölümü gizle.

Eğer ki VSCodium arayüzün küçükse "Ctrl" tuşu ile NumLock'taki "+" tuşuna basarak arayüzü büyütebilirsin.
(17 inçlik 1920x1080 çözünürlükteki bir ekranda, Linux'ta VSCodium'u 2 kez büyüterek kullanıyorum.)

VSCodium arayüzünü büyüttüysen ve eksi haline getirmek istiyorsan "Ctrl" tuşu ile NumLock'taki "0" tuşuna
basarak eski haline getirebilirsin.

Eğer ki VSCodium arayüzünü "Ctrl" ile "+" tuşlarına basarak büyütürsen alt çubuğun sağında
"Pencere Yakınlaştırma" bölümü gözükebilir. O bölüm gözükürse o bölümü de gizle.

Eğer ki bir dosya açtığında alt çubuğun sağında "Düzenleyici Girintisi" bölümü gözüküyorsa o bölümü gizle.

Eğer ki bir C dosyası açtığında alt çubuğun solunda "clangd (Uzantı)" bölümü gözüküyorsa o bölümü gizle.

Eğer ki bir Python dosyası açtığında sol çubukta "Test" menüsü varsa o bölümü gizle.

Son olarak; eğer ki VSCodium'da bir Java dosyasını, sağ üstteki "Run Java" butonuna basıp çalıştırdığında, alt
çubuğun solunda "Hata Ayıkla" bölümü gözüküyorsa o bölümü gizle.

Not1: VSCodium'da hiçbir uyarıda, "Bu kaynaktan bir daha uyarı gösterme" seçeneğine tıklama.

Not2:
    VSCodium'da run and debug butonunu gizleyip geri açamıyorsan sağ üstte üç nokta kısmına sağ tık yap ve
    "run and debug" seçeneğine tıkla.

Not3:
    Java'da debug ettikten sonra, çalıştırma butonu debug'da kaldıysa "Extension Pack for Java" eklenti
    paketini kaldır, VSCodium'u kapat. VSCodium'u tekrar aç ve Extension Pack for Java'yı tekrar kur.
